# Ganji HTML Project (간지 찾기 프로젝트)

## 프로젝트 소개
**Ganji HTML Project**는 왕의 치세년도에 따라 간지(60갑자)를 누구나 쉽게 찾을 수 있도록 돕기 위해 제작되었습니다. 
브라우저에서 파일을 실행하여 바로 사용할 수 있으며, GitHub Pages를 통해 웹페이지 형태로도 접속이 가능합니다.

**참고:** 이 프로젝트는 **데스크탑(PC)** 환경에 최적화되어 있습니다. 화면이 작은 휴대폰에서는 사용이 불편할 수 있으나, 화면이 큰 태블릿에서는 원활하게 작동합니다.

---

## 사용 방법

### 1. GitHub에서 파일 다운로드
1. 저장소 방문: [https://github.com/Sukju22/Ganji](https://github.com/Sukju22/Ganji)
2. `index.html` 파일을 다운로드합니다.
3. `Ganji_v`로 시작하는 버전 표시 파일들을 다운로드하여 사용해도 무방합니다. (내용은 `index.html`과 동일합니다.)

### 2. 브라우저에서 실행
- 다운로드한 `index.html` 파일을 더블 클릭하여 실행합니다.
- Chrome, Edge, Firefox, Safari 등 최신 브라우저를 권장합니다.

### 3. GitHub Pages (온라인 접속)
- 웹페이지 주소: [https://Sukju22.github.io/Ganji/](https://Sukju22.github.io/Ganji/)
- 위 주소는 항상 최신 버전의 `index.html`로 연결됩니다.
